# Aikido Security — Claude Code Plugin

Brings [Aikido Security](https://aikido.dev) directly into [Kiro Power](https://kiro.dev) via the Aikido MCP server. Scan code you write or modify for vulnerabilities and secrets.

## Installation and setup

Check out the [installation and setup guide](https://help.aikido.dev/ai-and-dev-tools/aikido-mcp/kiro-power-mcp) for detailed guidance on how to setup the Aikido Kiro Power.
